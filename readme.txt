A complete module to manage and display news on your site.

== Description ==

This plugin enables you to add, edit and delete news on your site. You can display these news on your site for visitors and also, you can control the delay time before the news is faded and replaced by the next news. Similarly, you can make the news active or inactive as well.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use 'Settings->News Publishing' screen to configure the plugin.
4. To display the news at a particular page or location, just write [CNP_HERE] at that location.
